# 书海拾贝 - 个人读书分享网站

基于Supabase和Netlify构建的个人读书分享平台，支持书籍浏览、书评查看和个人书单管理。

## 功能特性

- 📚 浏览书籍库，支持搜索和筛选
- ⭐ 查看其他用户的书评
- 📖 管理个人阅读书单（想读/在读/已读）
- 📱 响应式设计，支持移动端
- 🚀 基于现代Web技术栈

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **后端**: Supabase (PostgreSQL数据库)
- **部署**: Netlify
- **认证**: Supabase Auth (可扩展)
